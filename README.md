# Recipe Picker App

Basic App that can reads in recipe data and displays recipes in a slider

# To Build

clone ```https://github.com/HurricaneSKC/recipe-web-app.git```

Move to recipe-app directory
```
cd recipe-app
```

install dependencies
```
npm install
```

To run on local server

```
npm start
```
